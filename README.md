## 💻 Códigos Git

| Código | Função |
|--------|--------|
| git clone | Clona o repositório |
| git cd | Vai até a pasta desejada |
| git mkdir | Cria uma pasta na branch |
| git pull | "Puxa" as alterações do repositório |
| git push | "Empurra" as alterações do repositório |
